------------------------------------------------------------------------------
Running JOGRE Utilies
------------------------------------------------------------------------------

There are 2 currently utilities: -


1) JOGRE GAME GENERATOR
-------------------------------------------------------

    - Creates a blank game using templates.    
    - To run: -
    
        o Graphical application
        
            gamegengui.bat (windows)            
            gamegengui.sh  (linux)
            
        o command prompt 
        	
            java -jar gamegen.jar    (windows / linux)
   
   
2) JOGRE PROPERTY EDITOR
-------------------------------------------------------

    - Little app for editing jogre language labels
    - To run: -
    
        o Graphical application 
        
            gamegengui.bat (windows)
            gamegengui.sh  (linux)
